# Verdant Frontend - React + Vite

Admin dashboard for the Verdant lawncare platform.

## Quick Start

```bash
npm install
npm run dev  # Runs on http://localhost:3000
```

## Features

- Dashboard with business metrics
- Customer management
- AI-powered lead scoring
- React Router + TanStack Query
- Axios API client with auto authentication

## API Proxying

- Node.js API: `/api/v1` → `http://localhost:3001`
- Python AI: `/api/ai` → `http://localhost:8000`
